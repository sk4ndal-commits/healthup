# App Factory Template - Backend

This project serves as the base template for applications generated by the App Factory. It provides a robust, multi-tenant foundation with modern security and architecture practices.

## Functionality Contract

Any application based on this template is expected to implement and maintain a core set of functionalities. These are documented in detail in the [FUNCTIONALITY_CONTRACT.md](./FUNCTIONALITY_CONTRACT.md).

### Core Features at a Glance:
- **Multi-Tenancy**: Built-in account isolation.
- **Identity**: Registration, Login, and Password Recovery.
- **Security**: RBAC (Admin/User) and Dual-mode Auth (Cookie + JWT).
- **Localization**: Full support for German and English.
- **Auditability**: Integrated audit logging for critical actions.
- **Admin Tools**: User management and system monitoring.
- **API First**: Fully documented REST API with Swagger.

## Getting Started

### Prerequisites
- .NET 8 SDK
- A database provider (configured in `appsettings.json`)

### Running the Application
```bash
dotnet run --project src/Host/Host.csproj
```

### API Documentation
When running in development mode, you can access the Swagger UI at:
`https://localhost:<port>/swagger`
